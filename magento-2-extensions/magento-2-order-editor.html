Magento 2 Order Editor



The Milople Magento 2 Order Editor extension allows you to edit orders in the Magento 2 admin panel without having to cancel the order and create a new one. This can save you a lot of time and hassle, especially if you need to make minor changes to an order.

To use the Milople Magento 2 Order Editor extension, you will need to:

  1. Install the extension on your Magento 2 store.
  2. Configure the extension settings.
  3. Open the order that you want to edit.
  4. Click on the "Edit Order" button.
  5. Make the necessary changes to the order.
  6. Click on the "Save Order" button.

The Milople Magento 2 Order Editor extension allows you to edit a variety of order information, including:

  • Order status
  • Shipping address
  • Billing address
  • Payment method
  • Shipping method
  • Product quantities
  • Product prices
  • Order discounts
  • Order taxes

You can also use the Milople Magento 2 Order Editor extension to add or remove products from an order.

Here are some of the benefits of using the Milople Magento 2 Order Editor extension:

  • Save time and hassle by editing orders without having to cancel them and create new ones.
  • Reduce the number of order cancellations.
  • Improve customer satisfaction by making it easy to correct mistakes and make changes to orders.
  • Increase sales by making it easy to process orders with special requests.

Here are some tips for using the Milople Magento 2 Order Editor extension effectively:

  • Only edit orders if it is necessary.
  • Be careful when editing order prices and discounts.
  • Make sure to communicate with the customer before making any major changes to their order.
  • Keep a record of all order edits.

By following these tips, you can use the Milople Magento 2 Order Editor extension to save time and hassle, reduce order cancellations, improve customer satisfaction, and increase sales.
